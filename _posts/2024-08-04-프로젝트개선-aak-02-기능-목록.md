## 2. 메인 페이지
### 2.1 비회원
- 인덱스 페이지
  - 상단 네비게이션 바
    1. 로고 : 클릭 시 인덱스 페이지로 연결
    2. 강사페이지 링크 : 누르면 강사 로그인 페이지로 연결
    3. 강의 : 드롭다운으로 강의 카테고리
    4. 강사 : 강사 전체 목록 페이지로 연결
    5. 커뮤티니 : 드롭다운으로 공지사항, 이벤트, 문의
    6. 회원가입/로그인/로그아웃 : 로그인 전에는 회원가입/로그인으로 표시하고 로그인 후에는 로그인정보/로그아웃 표시
  - 배너 화면 : 현재 진행 중인 이벤트 표시, 신규 런칭 강의 혹은 신규 강사 표시, 없으면 기본 이미지 표시
  - 강의 목록
    1. 탭으로 강의 카테고리 별 전환
    2. 검색기능 : 강의 제목 혹은 강사명으로 검색
    3. 정렬기능 : 강의 제목, 강사명, 가격, 출시일, 수강신청수 수, 선호강의 수, 평점, 리뷰 수 오름차/내림차 정렬
- 강의 목록 페이지
- 강의 상세 페이지
- 공지사항
- 문의페이지
